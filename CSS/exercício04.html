Seletores

Seletor de TAG

Ao usar o seletor tipo TAG em H1 e H2 todos os elementos desse tipo vão herdar os estilos definidos pelo código CSS

Seletores tipo .ID

O seletor tipo #id deve ser usado uma única vez no arquivo! Cada elemento pode ter apenas um ID. Cada página pode ter apenas um elemento com aquele ID.

Seletor de Classe

Qualquer informação de estilo que precise ser aplicada a multiplos elementos em uma página deve ser feita em uma classe. Você pode usar a mesma classe para vários elementos. Você pode usar várias classes para um mesmo elemento
